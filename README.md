## Prerequisites

- **Git**
- **Python** - Version 3.14.X
- **UV** - package and virtual environment manager - <https://docs.astral.sh/uv/#the-pip-interface>

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/IgnacyBerent/EngineeringThesis.git
```

### 2. Install Dependencies

```bash
uv sync
```

### 3. Activate Virtual Environment

- Linux/Unix

```bash
source .venv/bin/activate
```

- Windows

```bash
.venv\Scripts\activate
```

### 4. Run

```bash
uv run python main.py
```
